# ระบบเบิกของ ห้องคลอด | โรงพยาบาลสะบ้าย้อย

เว็บแอปเช็คสต็อก/เบิกวัสดุ-เวชภัณฑ์ ทำงานเป็นไฟล์ HTML เดียว ไม่ต้องใช้เซิร์ฟเวอร์

## วิธีขึ้น GitHub Pages

1. สร้าง repository ใหม่บน GitHub (เช่น `stock-lr`)
2. อัปโหลดไฟล์ `index.html` เข้า repo (กด **Add file → Upload files**)
3. ไปที่ **Settings → Pages**
4. หัวข้อ **Build and deployment** → Source เลือก **Deploy from a branch**
5. Branch เลือก `main` โฟลเดอร์ `/ (root)` แล้วกด **Save**
6. รอสักครู่ จะได้ลิงก์ `https://<username>.github.io/<repo>/`

เปิดลิงก์บนมือถือได้เลย และ "Add to Home Screen" เพื่อใช้เหมือนแอป

## ข้อมูล

- **ฝ่ายการฯ** 76 รายการ (ตาม Google Sheet)
- **พัสดุ** 12 รายการ (ตัวอย่าง — แก้ไขได้ในไฟล์)
- **Lab** 12 รายการ (ตัวอย่าง — แก้ไขได้ในไฟล์)

## การแก้ไขรายการ

เปิด `index.html` แล้วแก้ที่ตัวแปร `RAW_FAIYAKAN`, `RAW_PASSADU`, `RAW_LAB`
รูปแบบ: `{ name: "ชื่อของ", unit: "หน่วย", req: จำนวนที่ต้องมี }`

## ฟีเจอร์

- นับจำนวนที่มีด้วยปุ่ม +/− → คำนวณ "ต้องเบิก" อัตโนมัติ
- สถานะ: ครบ / ไม่พอ / หมด
- ค้นหารายการ + แบ่ง 3 หมวด
- ส่งออกไฟล์ CSV (เปิดใน Excel ได้ รองรับภาษาไทย)
- บันทึกข้อมูลไว้ในเครื่อง (localStorage) — ปิดแล้วเปิดใหม่ข้อมูลยังอยู่

## หมายเหตุ

ข้อมูลถูกเก็บใน localStorage ของเบราว์เซอร์แต่ละเครื่อง ไม่ได้ sync ข้ามเครื่อง
หากต้องการฐานข้อมูลกลาง (หลายคนใช้ร่วมกัน) ต้องต่อ backend เช่น Google Sheets API หรือ Firebase เพิ่ม
